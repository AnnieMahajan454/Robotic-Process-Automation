# Web Scraping using UiPath (Mouse Activity)

This project demonstrates web scraping automation built with UiPath Studio.
The workflow opens Amazon in Chrome, searches for laptops, extracts tabular product data using mouse-based UI automation, and processes the results.

## Features

* Browser automation using Chrome activities
* Mouse-driven interaction with UI elements
* Table data extraction from web results
* Conversion of extracted data to text format
* Export of scraped data to Excel

## Workflow Overview

1. Launch browser and navigate to Amazon
2. Perform search interaction
3. Extract product listing table data
4. Store results in a DataTable
5. Output and save data for analysis

## Tools Used

* UiPath Studio
* Chrome Browser
* Excel Integration Activities
